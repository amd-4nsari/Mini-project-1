Mini Project 1 – Frontend Development

This is a frontend development mini project created as part of the Department of Computer Science & Engineering curriculum.
The project demonstrates the use of HTML, CSS, and JavaScript to build a responsive and interactive website.

🚀 Features

Responsive layout (works on desktop & mobile)

Navigation bar with smooth UI

Clean and modern design

Footer and structured sections

Interactive elements (buttons, menus, etc.)

🛠️ Technologies Used

HTML5 – Structure of the website

CSS3 – Styling & layout

JavaScript – Interactivity (if used)

VS Code – Code editor

Chrome DevTools – Debugging and testing
